Using the select and option tag

Notes:

If we obtain a reference to the select object; call this ref, then we can use it as follows:


ref.value gives us the value of selected option.

ref.selectedIndex provides the index of the option selected (starting at 0).

ref.options provides us with an array of the available options and we can then query each option with .value or .text. and .selected (true or false).

This article provides a reasonably clear guide.
